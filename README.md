# Login Form

Login Form Built Using HTML and CSS.


## Preview

![Login Form Screenshot](https://raw.githubusercontent.com/BinaryVortex/Login-Form-44/main/Screenshot%202024-07-06%20233454.png)


## About

A clean, responsive login form built with plain HTML and CSS. This project is a small starter/demo for practicing layout, form styling, and basic responsive techniques.


## Features

- Simple and modern login form layout
- CSS-only styling (no frameworks or JavaScript required)
- Ready to drop into any static site or use as a starter template


## Files

- `index.html` — the login form markup
- `style.css` — styling for the form
- `Screenshot 2024-07-06 233454.png` — preview image used in this README


## Usage

1. Clone or download the repository.
2. Open `index.html` in your browser (double-click the file or serve it from a local web server).

For a local server (optional):

- Python 3: `python -m http.server` and open http://localhost:8000


## Customization

- Colors, spacing, and fonts can be changed in `style.css`.
- Add form validation or integrate the form with your backend by adding JavaScript and updating the form `action`.
